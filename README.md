# Flask EC2 Deployment Project

## 📌 Project Overview
This is a production-style deployment of a Flask web application on AWS EC2 using Gunicorn and Nginx.

The project demonstrates how to deploy a Python web application to a real server with a proper reverse proxy and process manager.

---

## ⚙️ Tech Stack

- Python 3
- Flask
- Gunicorn
- Nginx
- AWS EC2 (Ubuntu)
- Git + GitHub

---

## 🏗️ Architecture

Client (Browser)
→ Nginx (Reverse Proxy)
→ Gunicorn (WSGI Server)
→ Flask Application
